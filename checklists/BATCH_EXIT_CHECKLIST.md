# Batch Exit Checklist

Sebelum lanjut batch berikutnya:

```text
[ ] Semua ticket batch selesai
[ ] Semua UAC terpenuhi
[ ] Unit tests batch pass
[ ] Integration tests yang relevan pass
[ ] Tidak ada out-of-scope implementation
[ ] Demo checkpoint berhasil
[ ] DECISIONS.md updated jika ada keputusan baru
[ ] BUG_LOG.md updated jika ada bug
[ ] Commit bersih
```
